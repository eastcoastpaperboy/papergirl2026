# Papergirl 2026

Retro web game prototype inspired by Paperboy-style gameplay.

## Run locally

1. Open a terminal in this project folder.
2. Start a local PHP server:

```bash
php -S localhost:8000
```

3. Open your browser:
   - Main page: `http://localhost:8000/`
   - City select: `http://localhost:8000/city_select.html`
   - Houston game: `http://localhost:8000/houston.html`

## Controls

- `ArrowLeft` / `ArrowRight` or `A` / `D`: move bike
- `ArrowUp`: jump
- `Space`: throw paper
- `P`: pause/resume
- `B`: back to city select
